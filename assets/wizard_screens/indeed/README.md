# Indeed API Setup Screenshots

Place step-by-step screenshots here to guide users through the Indeed setup.

## Naming Convention
`step_01.png`, `step_02.png`, etc.

## Captions File
Create `captions.json`:
```json
{
  "step_01": "Description of step 1",
  "step_02": "Description of step 2"
}
```

## Last Verified
Create `last_verified.txt` with the date screenshots were last confirmed accurate.
